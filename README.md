# my-resume

My personal résumé website — a responsive single-page application built with React, TypeScript and Vite, hosted for free on GitHub Pages.

Visit it live at [gustavorizzo.github.io/my-resume](https://gustavorizzo.github.io/my-resume/).

## Running locally

You will need *Node.js* installed. Then run:

```shell
npm install
npm run dev
```

The site content lives in [src/data/data.json](src/data/data.json).

## Deployment

Deployment is automated via GitHub Pages. The [workflow](.github/workflows/deploy.yml) builds and publishes the site whenever the `main` branch is updated.
